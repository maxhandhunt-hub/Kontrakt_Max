Финальный лендинг

1. Откройте проект через локальный сервер, а не через file://
   Пример:
   python -m http.server 8000

2. Откройте:
   http://localhost:8000

3. Google Apps Script уже прописан в app.js.

- index.html
- styles.css
- app.js
- positions.js
- cities.js
- Code.gs
- images/soldier1..10.webp
